# Stellaris-Leader-Command-Generator
A powerful desktop application for generating custom leader commands for the game Stellaris. It provides an intuitive GUI to select traits, configure advanced leader properties, and instantly produce ready-to-use console commands or event effects.  群星领袖指令生成器 – 一个功能强大的桌面工具，用于为游戏《群星》生成自定义领袖指令。通过直观的图形界面，你可以轻松选择特质、配置高级领袖属性，并立即生成可直接用于控制台或事件的代码。
